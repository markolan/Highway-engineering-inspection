# Highway-engineering-inspection
本项目基于 InternLM 实现llama3和Tutorial相结合（https://github.com/InternLM/Tutorial
），致力于开发一款先进的公路工程检测大模型，旨在提高公路工程检测的效率和准确性。该模型利用人工智能技术和大数据分析，对公路工程的结构健康、材料性能、施工质量等方面进行全方位检测和评估。
